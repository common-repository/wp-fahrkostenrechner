== Description ==
Einfacher Fahrkostenrechner per Google Distance Matrix API und Shortcode für Unternehmen wie Taxi oder Kuriere in der Schweiz, Deutschland und Österreich. Nur in deutsch verfügbar. Calculate Prices for Driving.

== Installation ==
1. Lade das Plugin aus der Wordpress Plugin Seite herunter. Du kannst auch die ZIP von kmu-internetseiten.ch herunterladen und danach im Adminmenü deiner Wordpress
    Installation das Plugin unter "Plugins" > "Installieren" > "Plugin hochladen" hochladen.
2. Aktiviere das Plugin im Pluginmenu in Wordpress

Was wird benötigt?

Plugin WP Fahrkostenrecher
API-Key von Google Distance Matrix API
den nachfolgenden Shortcode zum Einbinden des Fahrkostenkalkulator

Der Shortcode:

[KMUFahrkostenrechner apikey=YOUR_API_KEY fpland=CH fpwaehrung=CHF fptag=2.00 fpnacht=4.00 fptext=“Das ist einfach mal ein Test von WP Fahrkostenrechner“]

apikey= Ihr API-Key von Google Distance Matrix API
fpland= Kürzel für Ihr Land (CH,DE,AT)
fpwaehrung= Währung Ihres Landes (CHF,EUR)
fptag= Preis pro KM für Tagesfahrten
fpnacht= Preis pro KM für Nachtfahrten
fptext= „In Anführungszeichen gesetzter Text unterhalb des Resultates für Informationen zu Aufschläge, Bedingungen und Co.“

Eine Anleitung findest du unter: <a href="https://webseitenhilfe.ch/tipps-tricks/wordpress-plugin-wp-fahrkostenrechner/">hier</a>
